# arch-movie

## This repository will represents my master thesis. There are following packages in this project:
- :common - all common things of the project.
- :mvp - MVP representation of the project.
- :mvvm - MVVM representation of the project.
- :mvi - MVI representation of the project.

 

## There are following libraries which I use:
Jetpack AppCompat, Jetpack Core, Material Components, Constraint Layout, Swipe Refresh Layout, Lifecycle, Navigation, Fragment, Coil, Coroutines, FlowBinding, Koin, Room, Retrofit, OkHttp.
